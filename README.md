# Skills

Custom skills for [Claude Code](https://claude.ai/code).

## Installation

```bash
npx skills@latest add MHB2011/skills/<skill-name>
```

## Skills

### Planning

| Skill | Description |
|-------|-------------|
| [draft-prd](./draft-prd) | Create a PRD through user interview and save it to Ralph's local backlog |
| [draft-plan](./draft-plan) | Turn a local backlog PRD into a phased plan using tracer-bullet vertical slices |

### Tooling & Setup

| Skill | Description |
|-------|-------------|
| [setup-ralph](./setup-ralph) | Set up Ralph autonomous coding loops (HITL + AFK) with GitHub or Local mode |
